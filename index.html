코끼리 야바위

이벤트 설명

  1. 같은 커마와 장비투영의 루가딘 코끼리 운영진 3명
  2. 그 중 1명이 코끼리 탈을 벗은 뒤 다시 착용
  3. 코끼리들이 랜덤하게 돌아다니면서 서로 셔플
  4. 다 섞인 뒤에 처음에 탈을 벗었던 운영진을 선택하면 성공
  5. 최후 3인이 남을 때까지 게임 계속 진행
  6. 닉네임으로 구별하는걸 방지하기 위해, HUD를 숨긴 운영진의 디코 화면 공유로 게임을 진행

우승 상금

  • 최후 3인에게 길 지급 (남은 인원 중 한 명이라도 경쟁 원할 시, 게임 계속 진행)
  • 전원 탈락 시, 탈락자 포함 전체 인원 처음부터 재진행
  • 야바위 게임이므로 상금도 야바위
전체 상금 0 ~ 8,991,000 길
최후 3인 이하 인당 [ 9,000 길 / 남은 인원 수 X 대화주사위 (0 ~ 999) ]
예시 (인당 상금) 최후 3인 3,000 길 X (0 ~ 999) = 0 ~ 2,997,000 길
최후 2인 4,500 길 X (0 ~ 999) = 0 ~ 4,495,500 길
최후 1인 9,000 길 X (0 ~ 999) = 0 ~ 8,991,000 길

OX 퀴즈

이벤트 설명

  1. 운영진이 내는 퀴즈를 듣고 O / X 중 선택
  2. 최후 3인이 남을 때까지 게임 계속 진행

우승 상금

  • 최후 3인에게 길 지급 (남은 인원 중 한 명이라도 경쟁 원할 시, 게임 계속 진행)
  • 전원 탈락 시, 탈락자 포함 전체 인원 처음부터 재진행
전체 상금 6,000,000 길
최후 3인 이하 인당 [ 전체 상금 6,000,000 길 / 남은 인원 수 ]
예시 (인당 상금) 최후 3인 인당 2,000,000 길
최후 2인 인당 3,000,000 길
최후 1인 6,000,000 길

무작위 레벨링 타임어택

이벤트 설명

  1. 이벤트 참가자 랜덤 4인 1팀으로 파티 구성 (사전 구성 절대 불가, 형평성에 따라 고인물, 뉴비 등 인원 재분배될 수 있음)
  2. 인원 수가 부족한 파티는 운영진이 보충
  3. 이벤트 참가자 전부 파티 구성 완료 시, 무작위 레벨링 매칭
  4. 가장 빠르게 클리어한 파티 순으로 상품 지급
  5. 던전 외 토벌전(이프리트, 타이탄, 가루다 등)이 걸리면 클리어 후 다시 시작
  6. 던전 완료 시간과 파티원 전원의 목록이 함께 찍힌 스크린샷 제출 필수
